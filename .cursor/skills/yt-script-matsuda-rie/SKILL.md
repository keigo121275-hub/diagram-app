## スコープ（必ず守る）

- **対象**：**台本まで**（撮影・編集の細部指示は含めない）。
- **含めない**：**概要欄・タグ・固定コメントなどメタデータの文案執筆**、投稿後の分析レポート専用の執筆。
- **含める**：**台本に効かせるための**キーワード選定、YouTube検索結果の整理（ユーザーと対話）、構成案。
- **タイトル・サムネ**：候補と整理まで。**最終決定はユーザー。**
- **品質**：AIはたたき台。**壁打ちで詰めてから次へ。**

## 作業の流れ（1ステップずつ OK 待ち）

### A. 企画〜構成案（対話の正本は別ファイル）

`reference-script-structure-dialogue.md` の **フェーズ1→10** に沿う。**見出し単位で**進め、まとめ投げしない。

要約：目的 → SEO KW → 検索結果分析 → ペルソナ（複数可）と企画の狙い → 競合タイトル・順張り/逆張り企画 → タイトル・サムネ（候補〜**ユーザー決定**）→ 冒頭の声かけ → **PASTOR（A＝Amplify）** で骨格 → 章ごとのSEO意識 → アクション整理 → **具体事例** → **構成案**。PASTORの意味は **`reference-script-structure-dialogue.md`** の表のみ参照。
